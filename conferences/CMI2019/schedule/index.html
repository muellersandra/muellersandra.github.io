Abstracts for the parallel talks:

Dan Saattrup Nielsen, How does it all fit together?
Since there are many different technical aspects to the core model induction, it might be hard to see the big picture of what's actually going on, and how these technical pieces fit together. I'll attempt to give a bird's eye view of the core model induction at least up to $AD^L(\mathbb{R})$ and, maybe, push the intuition a tad further as well. As this is specifically a non-technical talk there will be no proofs, but many pictures.

William Chan, Descriptive Set Theory and Determinacy
Prewellorderings, scales, periodicity theorems, partition properties. Accessible with very minimal background in set theory or descriptive set theory.
